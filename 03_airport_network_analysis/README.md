# Airport Network Analysis (USA Flight Network)

## Objective
Analysis of the structure and dynamics of the US airport network over a 5-year period using graph theory techniques.

---

## Dataset
Flight data including connections between US airports over multiple years.

---

## Methodology

### Network Construction
- Representation of airports as nodes and flights as edges
- Construction of temporal network over 5 years

### Network Analysis
- Identification of key airports (central nodes)
- Analysis of most frequent routes
- Community detection in the network

### Robustness Analysis
- Simulation of network failures by removing key nodes
- Evaluation of network fragmentation and resilience

### Visualization
- Graph-based visualization of airport connectivity
- Geographical representation of network structure

---

## Technologies
R, RMarkdown, muxViz

---

## Key Outcomes
- Identification of critical hubs in the US airport network
- Understanding of network robustness and vulnerability
- Detection of structural communities in air traffic flows
