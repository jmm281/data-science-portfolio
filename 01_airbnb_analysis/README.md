# Airbnb Data Analysis & Recommendation System

## Objective
This project focuses on analyzing Airbnb property data to extract insights about revenue trends and to develop a recommendation system that suggests improvements to increase property performance.

---

## Dataset
Multiple datasets containing:
- Daily and monthly revenue data
- Property characteristics (location, price, amenities, etc.)

---

## Methodology

### Data Processing
- Data cleaning and handling missing values
- Integration of multiple datasets
- Feature engineering (creation of new informative variables)

### Exploratory Data Analysis
- Seasonal revenue trends analysis
- Identification of demand patterns
- Analysis of property features influencing income

### Recommendation System
- Development of a similarity-based recommendation system
- Comparison between properties to suggest improvements in amenities and features

### Visualization
- Data visualization to support insights and decision making

---

## Technologies
Python, Pandas, Scikit-learn, Matplotlib, Seaborn

---

## Key Outcomes
- Identification of seasonal income patterns
- Understanding of key drivers of property performance
- Functional recommendation system for property improvement suggestions
